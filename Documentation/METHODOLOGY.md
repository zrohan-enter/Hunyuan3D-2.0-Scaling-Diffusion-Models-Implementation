# Methodology

This project follows the two-stage Hunyuan3D 2.0 pipeline.

## Stage 1: Shape Generation

The input image is processed by Hunyuan3D-DiT to generate a bare 3D mesh.

## Stage 2: Texture Generation

The generated mesh can be passed into Hunyuan3D-Paint to generate texture.

## Scope

This project reproduces inference using released pretrained models. It does not train the model from scratch.
